# Лабораторная работа №7 — Архитектура и DDD-lite

Проект реализует упрощённую систему оплаты заказа
с использованием слоистой архитектуры и принципов DDD-lite.

## Архитектура
- Domain — доменная модель и бизнес-правила
- Application — use-case оплаты заказа
- Infrastructure — реализации интерфейсов
- Tests — модульные тесты без базы данных

## Результат
Бизнес-логика изолирована от инфраструктуры,
инварианты домена соблюдены,
use-case покрыт тестами.
